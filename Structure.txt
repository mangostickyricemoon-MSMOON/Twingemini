1. App.tsx
Root ของโปรเจกต์

รวมทุก component เข้ามาใช้งาน

จัดการ state หลัก เช่น cards, flipped, matched, showModal ต่าง ๆ

Render <Header />, <StatusBar />, <CardGrid />, <Footer />, และ modal ต่าง ๆ

2. components/Header.tsx
โลโก้ + wallet status + announcement button

ใช้ props: isConnected, address, setShowAnnouncement, playHoverSound

3. components/StatusBar.tsx
แสดง turn left, match count

ปุ่ม toggle เสียง 🔇/🔊

ใช้ props: picksTotal, picksAvailable, isMuted, toggleMute, playHoverSound

4. components/CardGrid.tsx
แสดงการ์ดทั้งหมดใน grid

ใช้ props: cards, flipped, matched, handleFlip

Render <Card />

5. components/Card.tsx
การ์ดแต่ละใบ

ใช้ props: card, isFlipped, onFlip

Render mangoIcon หรือ card image

6. components/Footer.tsx
ปุ่ม navigation: Home, HowTo, Ticket, Item, Wallet

ใช้ props: setShowAdminModal, setShowHowToModal, setShowTicketModal, setShowItemModal, setShowWalletModal

7. modals/AnnouncementModal.tsx
Modal แรกที่โชว์ Welcome

ปุ่ม Close → เปิดเสียงพื้นหลัง

8. modals/AdminModal.tsx
ฟังก์ชัน Deposit, Withdraw, Withdraw All

ใช้ PoolBalance component

ปุ่ม Close

9. modals/TicketModal.tsx
เลือกตั๋ว (ราคา + picks)

ปุ่ม Buy, Back

10. modals/GuardModal.tsx
แสดงข้อความ guardMessage

ปุ่ม OK

11. modals/HowToModal.tsx
อธิบายวิธีเล่นเกม

Ticket prices + rewards list

ปุ่ม Back

12. modals/ItemModal.tsx
แสดงข้อความ Coming Soon

ปุ่ม Close

13. modals/WalletModal.tsx
Connect / Disconnect wallet

ปุ่ม Close

14. modals/RewardModal.tsx
แสดงข้อความ Reward transferred

ปุ่ม OK → เปิด MascotModal

15. modals/MascotModal.tsx
แสดง Mascot Win/Lose

เล่นเสียง Win/Lose

ปุ่ม Close → เปิด TryAgain

16. components/TryAgain.tsx
ปุ่ม Play Again

ใช้ props: resetGame

17. hooks/useWallet.ts
Hook สำหรับจัดการ wallet

ฟังก์ชัน connect, disconnect, admin check

18. hooks/useContract.ts
Hook สำหรับ contract interaction

19. utils/contract.ts
ฟังก์ชัน getContract, CONTRACT_ADDRESS

20. utils/cards.ts
ฟังก์ชัน getRandomCards()

รับ cardImages → คืนค่า paired shuffled cards

21. utils/audio.ts
ฟังก์ชัน playHoverSound, playClickSound, toggleMute
